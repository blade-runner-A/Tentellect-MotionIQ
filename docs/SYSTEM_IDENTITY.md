# System Identity

## Chosen Name

**Tentellect MotionIQ**

## Positioning

**Tentellect MotionIQ** is a skeleton-first industrial safety intelligence system that enables robots and automation stacks to understand human posture, action, and risk in realtime.

## Components

- **MotionIQ Pipeline** (this repo): ingestion, skeleton extraction, gating, features, risk/action inference
- **MotionIQ Node** (hardware): on-site capture and edge inference device
- **MotionIQ Viewer** (`viewer-ui/`): operational visualization and QA interface

## One-line Description

Tentellect MotionIQ turns raw factory video into realtime worker-state intelligence robots can act on.
